TRIANGLE_SVG
Plot Triangle and Points with Scalable Vector Graphics (SVG)

TRIANGLE_SVG is a MATLAB library which uses Scalable Vector Graphics (SVG) to create a plot of a triangle and any number of points. Such plots can illustrate quadrature rules or sampling methods for the triangle.

Languages:

TRIANGLE_SVG is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:

TRIANGULATION_SVG, a MATLAB program which creates a Scalable Vector Graphics (SVG) image of a triangulation, which can be displayed by a web browser.

Reference:

  1. W3C, Scalable Vector Graphics 1.1 (Second Edition), 16 August 2011, http://www.w3.org/TR/SVG11/

Source Code:

  • r8_to_i4.m, maps X in [XMIN, XMAX] to integer IX in [IXMIN, IXMAX].
  • timestamp.m, prints the YMDHMS date as a timestamp.
  • triangle_svg.m, makes an SVG plot of a triangle and some points.

Examples and Tests:

  • triangle_svg_test.m, a sample calling program.
  • triangle_svg_test01.m, displays a 13 point quadrature rule in the unit triangle.
