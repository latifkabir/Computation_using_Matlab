FEM2D_SCALAR_DISPLAY_GPL
Surface Plot of FEM2D Data Using GNUPLOT

FEM2D_SCALAR_DISPLAY_GPL is a MATLAB program which reads datafiles describing a set of nodes, their triangulation, and the value of a scalar quantity at each node, and creates a graphics file of the corresponding surface plot, which can be displayed by GNUPLOT.

The program reads three data files:

  • prefix_nodes.txt, a node file, containing X, Y coordinates of points;
  • prefix_elements.txt, an element file, containing a list of sets of three points making up each triangular element; (if this file does not exist, the program will generate the element data automatically).
  • prefix_values.txt, a values file, containing the values of a scalar function U(X,Y) evaluated at each node.

Usage:

fem2d_scalar_display_gpl ( 'prefix' )
where 'prefix' is the common prefix for the node, element and value files:

The program creates two files:

  • prefix_data.txt, containing data;
  • prefix_commands.txt, containing commands for GNUPLOT.

To actually create the plot, type

gnuplot < prefix_commands.txt
which will create the file prefix.png

  • ell_nodes.txt, a node file;
  • ell_elements.txt, an element file;
