SPHERE_LLQ_GRID
Latitude/Longitude Quadrilateral Grids on a Sphere

SPHERE_LLQ_GRID is a MATLAB library which uses longitudes and latitudes to create grids of points, lines, and quadrilaterals over the surface of a sphere in 3D.

Languages:

SPHERE_LLQ_GRID is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.

SPHERE_LLT_GRID, a MATLAB library which computes a grid of triangles bounded by latitude and longitude lines over the surface of a sphere in 3D.
