AREA_UNDER_CURVE
Display Area Between Y=F(X) and X Axis.

AREA_UNDER_CURVE is a MATLAB function which displays the area under a curve, that is, the points (x,y) between the X axis and the curve Y=F(X).

Examples and Tests:

F1 is the function y=3/4 (1-x^2), between -1.5 and +1.5.

  • f1.m, evaluates the function.
  • f1.png, the plot created by the command: 
              area_under_curve ( -1.5, +1.5, @f1, 'Y=3/4(1-x^2)' )
