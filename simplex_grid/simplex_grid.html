SIMPLEX_GRID
Regular Grid Within a Simplex

SIMPLEX_GRID is a MATLAB library which generates a grid of points over the interior of a simplex in M dimensions.

Languages:

SIMPLEX_GRID is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.
