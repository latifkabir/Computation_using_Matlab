ELLIPSE_MONTE_CARLO
Monte Carlo Estimate of Integrals in an Ellipse

ELLIPSE_MONTE_CARLO is a MATLAB library which uses the Monte Carlo method to estimate the value of integrals over the interior of an ellipse in 2D.

ELLIPSE_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Source Code:

  • ellipse_area1.m, returns the area of an ellipse defined by a matrix.
  • ellipse_area2.m, returns the area of an ellipse defined by an equation.
  • ellipse_sample.m, samples points in an ellipse.
  • monomial_value.m, evaluates a monomial.
  • r8_uniform_01.m, returns a unit pseudorandom R8.
  • r8po_fa.m, factors an R8PO matrix.
  • r8po_sl.m, solves an R8PO system factored by R8PO_FA.
  • r8vec_normal_01.m, returns a unit pseudonormal R8VEC.
  • r8vec_uniform_01.m, returns a unit pseudorandom R8VEC.
  • timestamp.m, prints the current YMDHMS date as a time stamp.
  • uniform_in_sphere01_map.m, maps uniform points into the unit sphere.
