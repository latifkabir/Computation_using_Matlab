CC_IO
Read and Write Compressed Column (CC) Sparse Matrix Files

CC_IO is a MATLAB library which reads and writes compressed column (CC) sparse matrix files.

An MxN sparse matrix of NCC nonzero entries in compressed column format is described by three items:

  • ccc(), a list of N+1 compressed column indices.
  • icc(), a list of NCC row indices.
  • acc(), a list of NCC values.

The nonzero entries of column J are stored in locations CCC(J) through CCC(J+1)-1 of ICC and ACC.

Languages:

CC_IO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:

CC, a data directory which contains examples of the Compressed Column (CC) sparse matrix file format;

CC_TO_ST, a MATLAB library which converts a sparse matrix from compressed column (CC) to sparse triple (ST) format.

HB_IO, a MATLAB library which reads and writes sparse linear systems stored in the Harwell Boeing (HB) format for sparse matrices.

MM_IO, a MATLAB library which reads and writes sparse linear systems stored in the Matrix Market (MM) format.

ST_IO, a MATLAB library which reads and writes sparse linear systems stored in the Sparse Triplet (ST) format.

ST_TO_CC, a MATLAB library which converts sparse matrix data from Sparse Triplet (ST) format to Compressed Column (CC) format.

Source Code:

  • cc_data_read.m, reads data about a sparse matrix in CC format.
  • cc_header_read.m, reads header information about a sparse matrix in CC format.
  • cc_mv.m, multiplies a CC matrix times a vector.
  • cc_print.m, prints a sparse matrix in CC format.
  • cc_print_some.m, prints some of a sparse matrix in CC format.
  • cc_write.m, writes a sparse matrix in CC format to 3 files.
  • file_row_count.m, counts the number of row records in a file.
  • i4vec_data_read.m, reads data from an I4VEC file.
  • i4vec_dec.m, decrements an I4VEC.
  • i4vec_inc.m, increments an I4VEC.
  • i4vec_write.m, writes an I4VEC file.
  • r8vec_data_read.m, reads data from an R8VEC file.
  • r8vec_write.m, writes an R8VEC file.
  • timestamp.m, prints the current YMDHMS date as a time stamp.

Examples and Tests:

  • cc_io_test.m a sample calling program.
  • cc_io_test01.m writes a CC matrix to 3 files.
  • cc_io_test02.m reads a CC matrix from 3 files.
  • cc_io_test_output.txt, the output file.

Test #1 and #2 write and read back the information for the "simple" 5x5 matrix with 12 nonzero entries. There are files associated with this.

  • simple_acc.txt, the values.
  • simple_ccc.txt, the (0-based) compressed column indices.
  • simple_icc.txt, the (0-based) row indices.
