RNGLIB
A Package of Random Number Generators (RNG's) with Splitting Facilities

RNGLIB is a MATLAB library which implements random number generators (RNG's) which can generate one or more streams of random numbers.

RNGLIB is a portable set of software tools for uniform random variate generation. It provides for multiple generators running simultaneously, and each generator has its sequence of numbers partitioned into many long disjoint substreams. Simple procedure calls allow the user to make any generator jump ahead to the beginning of its next substream.

Related Data and Programs:

ASA183, a MATLAB library which implements the Wichman-Hill random number generator (RNG).

FAURE, a MATLAB library which computes elements of a Faure quasirandom sequence.

HALTON, a MATLAB library which computes elements of a Halton quasirandom sequence.

HAMMERSLEY, a MATLAB library which computes elements of a Hammersley quasirandom sequence.

MATLAB_RANDOM, MATLAB programs which illustrate the use of Fortran's random number generator (RNG) functions.

NIEDERREITER2, a MATLAB library which computes elements of a Niederreiter quasirandom sequence with base 2.

NORMAL, a MATLAB library which implements random number generators (RNG's) for normally distributed values.

PDFLIB, a MATLAB library which evaluates Probability Density Functions (PDF's) and produces random samples from them, including beta, binomial, chi, exponential, gamma, inverse chi, inverse gamma, multinomial, normal, scaled inverse chi, and uniform.

RANDLC, a MATLAB library which implements a random number generator (RNG) used by the NAS Benchmark programs.

RANLIB, a MATLAB library which produces random samples from Probability Density Functions (PDF's), including Beta, Chi-square Exponential, F, Gamma, Multivariate normal, Noncentral chi-square, Noncentral F, Univariate normal, random permutations, Real uniform, Binomial, Negative Binomial, Multinomial, Poisson and Integer uniform, by Barry Brown and James Lovato.

SOBOL, a MATLAB library which computes elements of a Sobol quasirandom sequence.

UNIFORM, a MATLAB library which implements random number generators (RNG's) for a variety of arithmetic types.

VAN_DER_CORPUT, a MATLAB library which computes elements of a van der Corput quasirandom sequence.

ZIGGURAT, a MATLAB program which generates points from a uniform, normal or exponential distribution, using the ziggurat method.

Source Code:

  • advance_state.m advances the state of a generator.
  • antithetic_get.m queries the antithetic value for a given generator.
  • antithetic_memory.m stores the antithetic value for all generators.
  • antithetic_set.m sets the antithetic value for a given generator.
  • cg_get.m queries the CG values for a given generator.
  • cg_memory.m stores the CG values for all generators.
  • cg_set.m sets the CG values for a given generator.
  • cgn_get.m gets the current generator index.
  • cgn_memory.m stores the current generator index.
  • cgn_set.m sets the current generator index.
  • get_state.m returns the state of generator G.
  • i4_uni.m generates a random positive integer.
  • ig_get.m queries the IG values for a given generator.
  • ig_memory.m stores the IG values for all generators.
  • ig_set.m sets the IG values for a given generator.
  • init_generator.m sets the state of generator G to initial, last or new seed.
  • initialize.m initializes the random number library.
  • initialized_get.m queries the INITIALIZED value.
  • initialized_memory.m stores the INITIALIZED value.
  • initialized_set.m sets the INITIALIZED value.
  • lg_get.m queries the LG values for a given generator.
  • lg_memory.m stores the LG values for all generators.
  • lg_set.m sets the LG values for a given generator.
  • multmod.m carries out modular multiplication.
  • r4_uni_01.m returns a uniform random real number in [0,1].
  • r8_uni_01.m returns a uniform random real number in [0,1].
  • set_initial_seed.m resets the initial seed and state for all generators.
  • set_seed.m resets the initial seed and the state of generator G.
  • timestamp.m prints the current YMDHMS date as a time stamp.
