POLYGON_MONTE_CARLO
Monte Carlo Estimate of Integrals Over Polygon in 2D

POLYGON_MONTE_CARLO is a MATLAB library which applies a Monte Carlo method to estimate the integral of a function over the interior of a polygon in 2D.

POLYGON_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.
