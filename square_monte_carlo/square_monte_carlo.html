SQUARE_MONTE_CARLO
Monte Carlo Estimate of Integrals Over Unit Square in 2D

SQUARE_MONTE_CARLO is a MATLAB library which estimates the integral of a function over the interior of the unit square in 2D.

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

SQUARE_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.

Source Code:

  • monomial_value.m, evaluates a monomial.
  • r8mat_uniform_01.m, returns a uniform pseudorandom R8MAT.
  • square01_area.m, returns the area of the unit square in 2D.
  • square01_monomial_integral.m, integrates a monomial in the unit square in 2D.
  • square01_sample.m, samples points from the unit square in 2D.
  • timestamp.m, prints the current YMDHMS date as a time stamp.
