TRIANGULATION_BOUNDARY_EDGES
Boundary Edges of a Triangulation

TRIANGULATION_BOUNDARY_EDGES is a MATLAB program which analyzes the triangulation of a region, identifies the edges that form the boundaries, organizes the boundary edges into one or more connected components, and writes a file describing the components and the sequence of boundary edges.

Either a 3-node or 6-node triangulation may be used.

Usage:

triangulation_boundary_edges ( 'prefix' )
where prefix is the common filename prefix:
  • prefix_nodes.txt contains the node coordinates,
  • prefix_elements.txt contains the element definitions.
  • prefix_boundary_edges.txt is a file created by the program. Each line lists a component index, and a pair of node indices.

Examples and Tests:

ELBOW is a curved region, looking like an elbow, that lies inside the unit square. It is triangulated using order 3 triangles. There are 320 nodes and 568 triangles. There are 66 nodes on the boundary.

  • elbow_nodes.txt, the node file.
  • elbow_elements.txt, the triangles.
  • elbow_output.txt, the printed output from a run of the sample program.
  • elbow_boundary_edges.txt, the boundary edge file.

ELL is an L-shaped region, using 96 triangles and 65 nodes. It is triangulated using order 3 triangles.

  • ell_nodes.txt, the node file.
  • ell_elements.txt, the triangles.
  • ell_output.txt, the printed output from a run of the sample program.
  • ell_boundary_edges.txt, the boundary edge file.

LAKE is an irregular lake-shaped region with an "island", which means the boundary is made of two components. It is triangulated using order 3 triangles.

  • lake_nodes.txt, the node file.
  • lake_elements.txt, the triangles.
  • lake_output.txt, the printed output from a run of the sample program.
  • lake_boundary_edges.txt, the boundary edge file.
