MATLAB_RANDOM_PARALLEL
Random Numbers in Parallel Matlab

MATLAB_RANDOM_PARALLEL is a directory of MATLAB programs which illustrate the use of Matlab's random number generators when using parallel programming features such as parfor or spmd.

Languages:

MATLAB_RANDOM_PARALLEL is available in a MATLAB version.

Related Data and Programs:

MATLAB_PARALLEL, examples which illustrate local parallel programming on a single computer with MATLAB's Parallel Computing Toolbox.

MATLAB_RANDOM, MATLAB programs which illustrate the use of Matlab's random number generator (RNG) functions.

Source Code:

  • matlab_random_parallel_test.m, the program.
  • matlab_random_parallel_test01.m, examines some simple features of rng() and rand();
  • matlab_random_parallel_test02.m, examines the use of rng() and rand() with spmd.
  • matlab_random_parallel_test03.m, examines the use of rng() and rand with parfor.
  • matlab_random_parallel_test_output.txt, the output file.
