XYL_DISPLAY
Display 2D Points and Lines in the MATLAB Graphics Window

XYL_DISPLAY is a MATLAB program which reads XYL information containing a list of 2D point coordinates, and the indices of points that form lines, and displays a plot in the MATLAB graphics window.

Usage:

xyl_display ( 'prefix' )
where
  • 'prefix' is the common file prefix
reads the point coordinate data in 'prefix.xy' and the line information in 'prefix.xyl', and displays a plot of the points and lines.

Examples and Tests:

ANNULUS defines a set of points and lines that outline a quadrilateral mesh of a semicircular annulus.

  • annulus.xy, the coordinates of the vertices.
  • annulus.xyl, the indices of vertices that form lines.
  • annulus.png, a PNG image of the plot.

SCPACK is a mesh defined by the SCPACK Scwarz-Christoffel mapping software.

  • scpack.xy, the coordinates of the vertices.
  • scpack.xyl, the indices of vertices that form lines.
  • scpack.png, a PNG image of the plot.
