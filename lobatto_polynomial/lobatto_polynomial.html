LOBATTO_POLYNOMIAL
Lobatto Polynomials

LOBATTO_POLYNOMIAL is a MATLAB library which evaluates the completed Lobatto polynomial and associated functions.

The completed Lobatto polynomial Lo(n,x) can be defined by: 

        Lo(n,x) = n * ( P(n-1,x) - x * P(n,x) )
where n is a positive integer called the order, x is a real value between -1 and +1, and P(n,x) is the Legendre polynomial.

The completed Lobatto polynomial Lo(n,x) has degree n+1, and is zero at x = -1 and x = +1.

Related Data and Programs:

CHEBYSHEV_POLYNOMIAL, a MATLAB library which considers the Chebyshev polynomials T(i,x), U(i,x), V(i,x) and W(i,x). Functions are provided to evaluate the polynomials, determine their zeros, produce their polynomial coefficients, produce related quadrature rules, project other functions onto these polynomial bases, and integrate double and triple products of the polynomials.

Source Code:

  • lobatto_polynomial_derivative.m, returns the derivative of the completed Lobatto polynomials Lo'(n,x).
  • lobatto_polynomial_plot.m, plots completed Lobatto polynomials Lo(n,x).
  • lobatto_polynomial_value.m, returns the value of the completed Lobatto polynomials Lo(n,x).
  • lobatto_polynomial_values.m, returns some tabulated values of the completed Lobatto polynomials Lo(n,x).
